## Resumen del proyecto

Innovatech Solutions es una plataforma de gestion de proyectos, recursos humanos y tareas. La EV3 incorpora el microservicio `ms-tareas`, responsables, estados, avance, visto bueno y reportes KPI, integrados con el frontend React, el BFF y el API Gateway.

La solucion incluye:

- Arquitectura de microservicios con Spring Boot.
- Frontend desarrollado en React.
- API REST y comunicacion mediante BFF y API Gateway.
- Persistencia independiente con bases de datos H2.
- Ejecucion integrada mediante Docker Compose.
- Pruebas unitarias backend con JUnit 5 y Mockito.
- Cobertura JaCoCo superior al minimo requerido del 60% en todos los componentes.
- Analisis de calidad con SonarQube.
